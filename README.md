## бһҳбһ»бһҒбһ„бһ¶бһҡбһ…бһҳбҹ’бһ”бһ„бҹ— (Features)
- рҹ•’ **бһ”бһҳбҹ’бһӣбҹӮбһ„бһҳбҹүбҹ„бһ„бһ”бҹ’бһҡбһҖбһҪбһҸбһ‘бҹ…бһҮбһ¶бһҳбҹүбҹ„бһ„бһ“бҹ…бһҖбһҳбҹ’бһ–бһ»бһҮбһ¶**бҹ– бһ”бһ„бҹ’бһ бһ¶бһүбһҳбҹүбҹ„бһ„бһ”бҹ’бһҡбһҖбһҪбһҸбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһ“бҹ…бһҖбһҳбҹ’бһ–бһ»бһҮбһ¶ бһ“бһ·бһ„бһҖбһ¶бһҡбһҡбһ¶бһ”бҹӢбһҗбһҷбһҖбҹ’бһҡбҹ„бһҷбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһҖбһҪбһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ”
- вҡҪ **Live Match Engine & Simulator**бҹ– бһҸбһ¶бһҳбһҠбһ¶бһ“бһҖбһ¶бһҡбһ”бҹ’бһҡбһҖбһҪбһҸбһ•бҹ’бһ‘бһ¶бһӣбҹӢбҹ— бһӮбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ¶бһӣбҹӢ бһўбҹ’бһ“бһҖбһҹбҹҠбһ»бһҸбһ”бһүбҹ’бһ…бһјбһӣбһ‘бһё бһ“бһ·бһ„бһҖбһ¶бһҸбһӣбһҝбһ„/бһҖбҹ’бһҡбһ бһҳ бһ–бҹ’бһҡбһҳбһ‘бһ¶бҹҶбһ„бһҳбһ¶бһ“бһҳбһ»бһҒбһ„бһ¶бһҡ Simulator бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бҹ’бһҡбһҖбһҪбһҸбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бҹ”
- рҹ“Ҡ **бһҸбһ¶бһҡбһ¶бһ„бһ–бһ·бһ“бҹ’бһ‘бһ»бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· (Standings)**бҹ– бһӮбһҺбһ“бһ¶бһҸбһ¶бһҡбһ¶бһ„бһ–бһ·бһ“бҹ’бһ‘бһ»бһ“бҹғбһ–бһјбһӣбһ‘бһ¶бҹҶбһ„ бҹЎбҹў (Group A бһҠбһӣбҹӢ L) бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһ¶бһҡбһ”бҹ’бһҡбһҖбһҪбһҸбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢ бһ¬бһҖбҹӮбһ”бҹ’бһҡбҹӮбҹ”
- рҹ“қ **Admin Panel**бҹ– бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһҖбҹӮбһ”бҹ’бһҡбҹӮбһ–бһ·бһ“бҹ’бһ‘бһ»бһҖбһ¶бһҡбһ”бҹ’бһҡбһҖбһҪбһҸбһҠбҹ„бһҷбһҠбҹғ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһЎбһҫбһ„бһңбһ·бһүбҹ”
- рҹ“ұ **Premium UI & Responsive Design**бҹ– бһҡбһ…бһ“бһ¶бһ”бһҗбһ„бһ„бһ№бһҸ (Sleek Dark Theme) бһҮбһ¶бһҳбһҪбһҷ Glassmorphism бһ“бһ·бһ„бһ‘бһ„бҹӢбһҮбһ¶бһҸбһ·бһ”бҹ’бһҡбһ‘бҹҒбһҹбһ“бһёбһҳбһҪбһҷбҹ—бҹ”

## бһҡбһ”бҹҖбһ”бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё (How to Run)
1. бһҠбҹҶбһЎбһҫбһ„ Node.js (бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһҳбһ¶бһ“)
2. бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹҶбһЎбһҫбһ„ dependenciesбҹ–
   ```bash
   npm install
   ```
3. бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ dev serverбҹ–
   ```bash
   npm run dev
   ```
4. бһ”бһҫбһҖ Browser бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһўбһ¶бһҹбһҷбһҠбҹ’бһӢбһ¶бһ“бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһү (бһ§. `http://localhost:5173`)

## бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһұбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбһҫ Netlify (Deploy to Netlify)

1. бһ”бһ„бҹ’бһҖбһҫбһҸбһӮбһҺбһ“бһё бһ“бһ·бһ„бһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢ [Netlify](https://www.netlify.com/)
2. бһ—бҹ’бһҮбһ¶бһ”бҹӢбһӮбһҺбһ“бһё Git (GitHub, GitLab, etc.) бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹӮбһӣбһҳбһ¶бһ“бһӮбһҳбҹ’бһҡбҹ„бһ„бһ“бҹҒбҹҮ
3. бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҷбһҖбһӮбһҳбҹ’бһҡбҹ„бһ„ `FIFA2026` бһ“бҹҒбҹҮ
4. Netlify бһ“бһ№бһ„бһҡбһҖбһғбһҫбһүбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҖбҹ’бһ“бһ»бһ„ `netlify.toml` бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбҹ–
   - **Build command**: `npm run build`
   - **Publish directory**: `dist`
5. бһ…бһ»бһ… **Deploy site** бһҮбһ¶бһҖбһ¶бһҡбһҹбҹ’бһҡбҹҒбһ…бҹ”
